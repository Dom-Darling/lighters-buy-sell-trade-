# Lighters Buy, Sell, Trade Website

This repository contains the code for a buy, sell, and trade website dedicated to lighters. The website aims to facilitate the exchange of lighters among users, providing a platform for those looking to acquire, donate, or barter their lighters.

## Getting Started

To get started with the website, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/your-username/lighters-buy-sell-trade.git
   ```

2. Navigate to the project directory:
   ```
   cd lighters-buy-sell-trade
   ```

3. Install the required dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

## Features

The website offers the following features:

- User registration and authentication
- Listing of available lighters for sale or trade
- Search functionality to filter lighters by brand, type, or condition
- User profiles to display personal information and listings
- Messaging system for communication between users
- Rating and review system for users
- Admin panel for managing listings, users, and transactions

## Instructions

To use the website, follow these steps:

1. Register an account by providing your email address and a password.
2. Log in to your account.
3. Browse the available lighters by clicking on the "Browse Lighters" link in the navigation bar.
4. Filter the lighters by brand, type, or condition using the search bar and dropdown menus.
5. Click on a lighter to view its details and contact the seller.
6. If you want to list your own lighter for sale or trade, click on the "List a Lighter" link in the navigation bar.
7. Fill out the form with the details of your lighter and submit it.
8. Once your listing is approved by an admin, it will be visible on the website.
9. To manage your listings, visit your user profile by clicking on your username in the navigation bar.
10. Click on the "My Listings" tab to view and edit your active listings.
11. To message other users, click on their profile and send them a message.
12. To rate and review other users, visit their profile and leave a rating and review.
